【小说开头使用说明】

请将你的小说开头文本放入此目录。

文件命名规则：
- novel-{novel-id}-start.txt

示例：
- novel-my-story-start.txt

开头要求：
- 包含小说标题
- 包含简介
- 包含第一章或开头内容
- 建议长度 1000-3000 字

示例格式：
```
【标题】我的小说
【类型】都市
【简介】一个普通人的不平凡故事

【第一章】

清晨的阳光透过窗帘的缝隙洒进房间，林默睁开眼睛，看着天花板上那道熟悉的光斑。

又是新的一天，又是重复的生活。他这样想着，却不知道，今天的遭遇将彻底改变他的人生轨迹...

（续写提示：主角即将遇到一个神秘人物，这个人将改变他的人生）
```
