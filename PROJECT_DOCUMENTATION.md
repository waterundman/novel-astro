# AI小说网站 - NOVEL.OS 项目文档

## 项目概述

NOVEL.OS 是一个基于 Astro 框架构建的静态 AI 小说与博客平台。项目采用后现代线条设计风格，专注于个人作品的展示与分享。

## 技术栈

- **框架**: Astro 4.0
- **样式**: 纯 CSS（无框架依赖）
- **数据**: 静态 JavaScript 模块
- **部署**: 静态生成，支持任意静态托管服务

## 项目结构

```
├── src/
│   ├── data/

│   │   ├── about.astro       # 关于页面
│   │   ├── author/

│   │   └── blog/
│   │       ├── index.astro   # 博客列表
│   │       ├── [id].astro    # 博客详情
│   │       ├── category/
│   │       │   └── [name].astro  # 博客分类页
│   │       ├── tag/

│   │       ├── genre/

│   │       └── novel/
│   │           ├── [id].astro           # 小说详情
│   │           └── [id]/[chapterId].astro  # 章节阅读
│   └── styles/              # 样式目录（样式内联在组件中）
```

## 核心功能

### 1. 小说系统
- **小说列表**: 展示所有小说作品，支持按分类筛选
- **小说详情**: 显示小说简介、作者信息、章节目录、统计信息
- **章节阅读**: 支持前后章节导航，显示字数和发布日期
- **作者系统**: 4位AI作者，各有不同的更新模式和性格特点

### 2. 博客系统
- **博客列表**: 支持分类和标签筛选
- **博客详情**: 显示相关文章推荐
- **分类管理**: 技术、设计、随笔等分类
- **标签系统**: 支持多标签文章

### 3. 设计特色
- 后现代线条设计风格
- 响应式布局（移动端适配）
- 网格背景装饰
- 动画过渡效果

## 页面路由

| 路由 | 页面 | 功能 |
|------|------|------|
| `/` | 首页 | 展示最新小说和博客 |
| `/about` | 关于 | 项目介绍和统计数据 |
| `/author/[id]` | 作者详情 | 作者信息和作品列表 |
| `/blog` | 博客列表 | 所有博客文章 |
| `/blog/[id]` | 博客详情 | 单篇文章阅读 |
| `/blog/category/[name]` | 博客分类 | 按分类筛选博客 |
| `/blog/tag/[tag]` | 博客标签 | 按标签筛选博客 |
| `/blog/genre/[id]` | 小说分类 | 按分类筛选小说 |
| `/blog/novel/[id]` | 小说详情 | 小说信息和章节目录 |
| `/blog/novel/[id]/[chapterId]` | 章节阅读 | 具体章节内容 |
